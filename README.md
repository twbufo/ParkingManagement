ParkingManagement
=================

任务分配：
一、需求1

		1、停车：
			(1)如果有人来停车，先查看停车场是否有空车位。
			(2)如果有空车位，登记车辆信息，发给司机停车卡，让司机停车到任何空车位，空车位个数减1，触发显示屏刷新。
			(3)如果停车场无空车位，拒绝停车。
		
		2、取车
			(1)如果有人来取车，先收卡，读取已停车的信息。
			(2)如果相应的停车场车位全为空，则不允许停车。
			(3)如果车卡一致，放行允许取车，空车位个数加1，触发显示屏刷新。


二、需求2

	停车仔：
		1、停车
			(1)如果有人来停车，先看所有管理的停车场中是否有空车位。
			(2)如果有空车位，登记车辆信息，发给司机停车卡，停车到任何空车位，空车位个数减1，触发显示屏刷新。
			(3)如果所有管理的停车场所都没有无空车位，拒绝停车。
		2、取车
			(1)如果有人来取车，先收卡，读取已停车的信息。
			(2)如果相应的停车场车位全为空，则不允许停车。
			(3)如果车卡一致，放行允许取车，空车位个数加1，触发显示屏刷新。
				
三、需求3
	聪明的停车仔：
		1、停车
			(1)如果有人来停车，先看所有管理的停车场中是否有空车位。
			(2)如果有空车位，登记车辆信息，发给司机停车卡，将车停到空车位最多的那个停车场，该停车场的空车位个数减1，触发显示屏刷新。
			(3)如果所有管理的停车场所都没有无空车位，拒绝停车。
		2、取车
			(1)如果有人来取车，先收卡，读取已停车的信息。
			(2)如果相应的停车场车位全为空，则不允许停车。
			(3)如果车卡一致，放行允许取车，该停车场的空车位个数加1，触发显示屏刷新。

四、需求4
	超级的停车仔：
		1、停车
			(1)如果有人来停车，先看所有管理的停车场中是否有空车位。
			(2)如果有空车位，登记车辆信息，发给司机停车卡，将车停到空置率最高的那个停车场，该停车场的空车位个数减1，触发显示屏刷新。
			(3)如果所有管理的停车场所都没有无空车位，拒绝停车。
		2、取车
			(1)如果有人来取车，先收卡，读取已停车的信息。
			(2)如果相应的停车场车位全为空，则不允许停车。
			(3)如果车卡一致，放行允许取车，该停车场的空车位个数加1，触发显示屏刷新。
